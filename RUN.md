# Run Instructions

## Downloading App

Download the appropriate app and run it.
- .dmg for MacOS
- .exe for Windows

## Creating an Account

Press the register button and create an account with a username and password. If you already have an account, you can just log in.

## Sending a Message

Create a chat by typing in someone else's username. Now, you can send them chats and read chats that have been sent to you. Feel free to send as many chats/create as many chats as you would like!
